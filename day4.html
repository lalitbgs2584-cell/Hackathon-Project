📂 File Paths in HTML

👉 File paths tell the browser where to find a resource, such as an image, CSS file, or another page.

There are two main types: 📁 Relative Paths and 🌍 Absolute Paths.

📖 Learn more at GeeksforGeeks

📁 Relative File Paths

✨ Relative paths depend on the current file’s location.

📌 Type 💻 Example 📖 Usage 🖼️ Image
📂 Same folder about.html ➡️ File in the same folder
📁 Sub-folder Images/logo.png ➡️ File inside a sub-folder
⬆️ Parent folder ../index.html ➡️ File one level up

📖 Detailed explanation: GFG - File Paths

🌍 Absolute File Paths

✨ Absolute paths use the full address of the resource.

📌 Type 💻 Example 📖 Usage 🖼️ Image
🌐 Full URL https://example.com/image.jpg ➡️ External resources on the web
💻 Local absolute path C:/Users/Username/Documents/index.html ⚠️ Works only on your computer

📖 Reference: Learn on GFG

🖼️ Image Paths

✨ Use the <img> tag with file paths to display images.

📌 Example 💻 Code 📖 Usage
<img src="images/painting.jpg" alt="Logo"> ➡️ From a sub-folder
<img src="https://images.unsplash.com/photo-1531366936337-7c912a4589a7" alt="Image"> ➡️ From an external URL

📖 Extra reading: Images in HTML - GFG

🔗 Link Paths

✨ Use the <a> tag with paths to navigate.

📌 Type 💻 Example 📖 Usage
📁 Relative link <a href="first.html">Day 1</a>
➡️ Day 1		 Go to a page in the same folder
🌍 Absolute link <a href="https://www.google.com">Google</a>
➡️ Google		 Go to an external site

✨ A multipage website is a collection of interconnected HTML pages.

Example navigation:
