📦 Nested Table

Code Output 
<table border="1">
  <tr><th>Main Header</th><th>Main Header</th></tr>
  <tr>
    <td>
      <table border="1">
        <tr><td>Nested 1</td><td>Nested 2</td></tr>
      </table>
    </td>
    <td>Main Cell 2</td>
  </tr>
</table>
Main Header Main Header
Nested 1Nested 2
 Main Cell 2

✅ Summary

  • <table>: Defines a table
  • <tr>: Defines a row
  • <th>: Table header cell
  • <td>: Table data cell
  • colspan: Merge cells horizontally
  • rowspan: Merge cells vertically
  • You can add images, links, and text inside any table cell.
